<h1 align="center">📱 Firestore CRUD V1 </h1>

<p align="center">
A simple Flutter application that uses Firestore to store and manage data for a demo CRUD notes application.
</p>

---

## .✦ ݁˖ Features

- Reactive UI using StreamBuilder
- Notes data stored in Cloud Firestore 
- Create new notes in a Cloud Firestore collection
- Edit existing notes from Cloud Firestore collection
- Delete notes from Cloud Firestore collection

---

## ˙⋆✮ How it works

The application uses Cloud Firestore to store notes data.
Adding a new note creates a document in `notes` collection.
Users can also edit fields inside existing documents or delete those documents.

---

## ⋆✿˖ Project Structure

```text                
lib/                     
├── firebase_options.dart   # Firebase configurations generated by FlutterFire
└── main.dart               # Application entry point and CRUD notes UI logic

pubspec.yaml                # Project dependencies (firebase_core, cloud_firestore, etc.)
```

---

## ⋆✿˖ Cloud Firestore Structure

```text                
Cloud Firestore > Database
└── notes                         # Collection that stores notes
       └── [notes_id]             # Document ID (auto-generate or custom)
                ├── title: "Example title" (String)
                ├── text: "Example note text" (String)
                └── createdAt: Timestamp
```
